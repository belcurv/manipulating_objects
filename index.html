Object manipulation

Introduction

This converts an array of objects to a different javascript object. The input array elements are objects that have a lot of common data. For example, there are a month's worth of dates (Feb 1 2016 through Feb 29 2016), and each source has sales of multiple fruits. The goal was to output a new object, where "target":"value" pairs were collected under shared sources and all those sources were collected under shared dates.
